If you're interested in pursuing interactive, web-based apps, pursuing a track with Ruby/Rails might be up your alley. Popular webapps written with Ruby on Rails include Github, Kickstarter, and Goodreads. If developing in these fields interests you, Rails is the way to go.

C#


C# is popular and widely used with major companies across the world, including Microsoft, Intel and Cisco. If you're ready to work collaboratively in a large, established corporate world, C# will help get you there.

React is a cutting edge library for Javascript that is used for building user interfaces. If you're looking for work with companies like Facebook, Instagram, Netflix, or Dropbox, ReactJS is a must.
